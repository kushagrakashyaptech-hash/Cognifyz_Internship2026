# Level 1 - Task 3: Geospatial Analysis

## Overview

This project performs geospatial analysis on a restaurant dataset as part of the Cognifyz Data Science Internship Program.

The objective of this task is to visualize restaurant locations using geographical coordinates, analyze restaurant distribution across different cities and countries, and investigate whether location has any relationship with restaurant ratings.

---

## Objectives

* Visualize restaurant locations using latitude and longitude data.
* Analyze restaurant distribution across different countries.
* Analyze restaurant distribution across different cities.
* Explore the relationship between restaurant location and aggregate rating.
* Perform basic correlation analysis between geographical coordinates and ratings.

---

## Dataset Information

* Dataset: Restaurant Dataset
* Total Records: 9551
* Total Features: 21
* Geospatial Features:

  * Latitude
  * Longitude

---

## Technologies Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

---

## Analysis Performed

### 1. Data Loading and Preprocessing

* Loaded the dataset using Pandas.
* Examined dataset structure and dimensions.
* Handled missing values in the "Cuisines" column.

### 2. Restaurant Location Visualization

* Used Latitude and Longitude coordinates to create a scatter plot.
* Visualized restaurant locations across different geographical regions.

### 3. Country Distribution Analysis

* Counted restaurants for each country code.
* Visualized restaurant distribution using bar charts.

### 4. City Distribution Analysis

* Identified cities with the highest number of restaurants.
* Created visualizations for the top cities in the dataset.

### 5. Rating and Location Analysis

* Calculated average restaurant ratings by city.
* Compared rating trends across different locations.

### 6. Correlation Analysis

* Computed correlation between Latitude and Aggregate Rating.
* Investigated whether geographical location influences restaurant ratings.

---

## Key Findings

* Restaurants are concentrated in specific geographical regions.
* Restaurant distribution varies significantly across countries and cities.
* Certain cities contain a much higher number of restaurants than others.
* Average restaurant ratings differ across locations.
* Geographical coordinates alone do not strongly predict restaurant ratings.
* Location may influence ratings indirectly through local restaurant quality and customer preferences.
