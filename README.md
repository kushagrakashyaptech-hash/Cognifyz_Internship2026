# Level 2 - Task 2: Price Range Analysis

## Overview

This project analyzes restaurant price ranges and their relationship with ratings.

## Objectives

- Determine the most common price range.
- Calculate average ratings for each price range.
- Identify the rating color associated with the highest average rating.

## Technologies Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Analysis Performed

- Price Range Distribution
- Average Rating by Price Range
- Rating Color Analysis
- Visualization of Rating Trends
