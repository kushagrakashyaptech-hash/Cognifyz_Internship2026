# Level 1 - Task 2: Descriptive Analysis

## Overview

This project performs descriptive analysis on a restaurant dataset as part of the Cognifyz Data Science Internship Program.

The objective of this task is to calculate statistical measures for numerical columns, analyze categorical variables, and identify the most common cuisines and cities in the dataset.

---

## Objectives

* Calculate basic statistical measures such as:

  * Mean
  * Median
  * Standard Deviation
  * Minimum and Maximum Values

* Explore the distribution of categorical variables:

  * Country Code
  * City
  * Cuisines

* Identify:

  * Top cities with the highest number of restaurants
  * Top cuisines with the highest number of restaurants

---

## Dataset Information

* Dataset: Restaurant Dataset
* Total Records: 9551
* Total Features: 21

---

## Technologies Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

---

## Analysis Performed

### 1. Data Loading and Preprocessing

* Loaded the dataset using Pandas.
* Checked dataset dimensions.
* Handled missing values in the "Cuisines" column.

### 2. Statistical Analysis

* Generated descriptive statistics for numerical columns using Pandas.
* Analyzed measures such as mean, standard deviation, minimum, maximum, and quartiles.

### 3. Country Code Distribution

* Examined the distribution of restaurants across different country codes.
* Visualized the distribution using bar charts.

### 4. City Distribution

* Identified cities with the highest number of restaurants.
* Visualized the top cities using bar charts.

### 5. Cuisine Distribution

* Analyzed the frequency of cuisine categories.
* Identified the most common cuisines in the dataset.

---

## Key Findings

* The dataset contains restaurants from multiple cities and countries.
* Certain cities have a significantly higher concentration of restaurants.
* North Indian cuisine is the most frequently occurring cuisine category.
* Several cuisine combinations are highly represented in the dataset.
* The restaurant distribution is not uniform across cities and cuisines.
